# TwitchEmotes Menu
Improves TwitchEmotes V2 with more organized menu. Developed for TBC-Classic.

# Requirements
- https://www.curseforge.com/wow/addons/twitch-emotes-v2
    - Tested with Twitch Emotes V2.43.1

# Usage
- Right-click the minimap button
- Open the chat
- Click on emote!
